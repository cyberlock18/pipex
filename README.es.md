## Lo que aprenderás / habilidades adquiridas
- Comprender el uso de tuberías en programas de C
- Implementar funciones para manejar cadenas y archivos
- Crear un Makefile para compilar el proyecto

## Comandos para construir y ejecutar

Para compilar el proyecto, utiliza los siguientes comandos:

```bash
make          # Compila el proyecto
make clean    # Elimina los archivos objeto y binarios generados
make fclean   # Elimina todos los archivos generados
make re       # Recompila el proyecto
```
