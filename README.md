## What You Learn / Skills Acquired:
- Understanding of Unix programming concepts
- Process handling and Inter-process communication (IPC)
- File handling and manipulation in a Unix environment
- Build systems using Makefiles

## Build & Run Commands:
To build the project, run:

```bash
make
```

To clean up the compiled object files, run:

```bash
make clean
```

To perform a full clean including the executable, run:

```bash
make fclean
```

To rebuild the project from scratch, run:

```bash
make re
```
